### Dependencies

- ```sudo apt-get install docker-compose```
- ```sudo apt-get install docker```

Add a .env inside front/
Inside the .env add the following key:

```REACT_APP_API_URL=```

For example ```REACT_APP_API_URL=http://0.0.0.0:8080```

### How to launch

```docker-compose build```
```docker-compose up```
